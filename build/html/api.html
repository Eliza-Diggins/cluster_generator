API#

utils

Core utilities for logging and computation throughout the library.

radial_profiles

Radial Profiles Radial profiles, as instantiated through the radial_profiles.RadialProfile class are wrappers for standard lambda functions which are used to provide additional structure for radial profiles of temperature, density, cumulative mass, and entropy in galaxy clusters.

particles

Module for management of particle initial conditions.

relations

Relations The functions in this module provide empirical relations between profile variables in a variety of cases.

gravity

Gravity The gravity module contains the gravity.Potential class, which essentially acts as a wrapper for solving the Poisson equation in each of the available gravitational theories. Potential objects contain fields and can be written to / read from binary files.
