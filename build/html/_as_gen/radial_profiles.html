radial_profiles#

Radial Profiles#

Radial profiles, as instantiated through the radial_profiles.RadialProfile class are wrappers for standard lambda functions which are used to provide additional structure for radial profiles of temperature, density, cumulative mass, and entropy in galaxy clusters.

Each RadialProfile object has attached methods for altering the profile to add cores, truncate the profile, and a variety of other tasks.
