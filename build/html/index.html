The Cluster Generator Project#

The Cluster Generator Project (CGP) is a cross-platform Python library for generating initial conditions of galaxy clusters for N-body / hydrodynamics codes. CGP provides a variety of construction approaches, different physical assumption, profiles, and gravitational theories. Furthermore, the CGP is intended to interface with the vast majority of N-body / hydrodynamics codes, reducing the headache of converting initial conditions between formats for different simulation software. GCP’s goal is to provide comprehensive tools for modeling and implementation of galaxy clusters in astrophysical simulations to promote the study of galaxy cluster dynamics.

This repository contains the core package, which is constructed modularly to facilitate easy development by users to meet particular scientific use cases. All of the necessary tools to get started building initial conditions are provided.

You can access the documentation here, or build it from scratch using the ./docs directory in this source distribution.

Development occurs here on Github, if you encounter any bugs, issues, documentation failures, or want to suggest features, we recommend that you submit an issue on the issues page of the repository.

For installation directions, visit the getting started page.

Radial Profiles

Each of the following radial profiles is included for use in cluster construction:

NFW Profiles

  • nfw_density_profile()

  • nfw_mass_profile()

  • snfw_density_profile()

  • snfw_mass_profile()

  • cored_snfw_density_profile()

  • cored_snfw_mass_profile()

  • tnfw_density_profile()

  • tnfw_mass_profile()

Hernquist Profiles

  • hernquist_density_profile()

  • hernquist_mass_profile()

  • cored_hernquist_density_profile()

  • cored_hernquist_mass_profile()

  • einasto_density_profile()

  • einasto_mass_profile()

General Profiles

  • power_law_profile()

Paper Specific Profiles

  • Vikhlinin et. al. 2006 - vikhlinin_density_profile() - vikhlinin_temperature_profile()

Entropy Profiles

  • walker_entropy_profile()

  • baseline_entropy_profile()

  • broken_entropy_profile()

Gravitational Theories

The CGP provides not only high quality initial condition generation capacity, but also provides a comprehensive catalog of alternative gravity theories to explore. The following are built-in, but adding more is a relatively simple task:

  • Newtonian Gravity

MONDian Gravities

  • AQUAL

  • QUMOND

Implemented Codes

The CGP provides end-to-end initial condition generation tools for all of the following codes:

  • RAMSES

  • ATHENA++

  • AREPO

  • GAMER

  • FLASH

  • GIZMO

  • ENZO

Available Datasets

The Collections system provides users access to pre-built galaxy clusters from the available literature. Cluster fits are available for all of the following papers:

  • Vikhlinin et. al. 2006

Automated Non-Physicality Correction

The CGP provides a purpose built algorithm for non-physical corrections in initialized clusters to reduce labor overhead in the generation of the initial conditions. For more information, visit the Correcting Non-Physical Behavior page.

Related Projects#

PyXSIM

Convert your CGP generated systems into synthetic photon event lists for use in simulating observations using X-ray observatories using PyXSIM. the CGP is designed to easily interface with this library to provide as much ease as possible when building simulated observations of clusters.

Documentation

SOXS

Coupled with PyXSIM, SOXS is a instrument simulation tool for turning mock photon counts into realistic X-ray observations specific to the behavior of specific instruments like CHANDRA, XMM-Newton, and NuSTAR.
